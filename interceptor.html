Interceptor Pattern

What is Interceptor ? ===================== Interceptor is an Architectural Pattern which can replace or augment * Wrapper * Decorator * Facade * Adapter * Proxy * Observer * Strategy * Chain of Command * Mediator * Filters * ServiceLocator * Front Controller * Delegation * Static AOP That's actually quite a lot ! This is mainly because it's an Architectural Pattern which are compressive in nature. It looks like this, ``` class Interceptor { public intercept(Invocation v); } ``` Essentially it is an indirection at the call level.
